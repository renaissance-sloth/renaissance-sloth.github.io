# append

Adds the specified string to the end of another string.

{% raw %}
```
//input
{{ "/my/fancy/url" | append: ".html" }}
```
{% endraw %}

```
//output
{{ "/my/fancy/url" | append: ".html" }}
```

`append` can also accept a variable as its argument.

{% raw %}
```
//input
{% assign filename = "/index.html" %}
{{ "website.com" | append: filename }}
```
{% endraw %}

```
//output
{% assign filename = "/index.html" %}
{{ "website.com" | append: filename }}
```
