# abs

Returns the absolute value of a number.

{% raw %}
```
//input
{{ -17 | abs }}
{{ 4 | abs }}
```
{% endraw %}

```
//output
{{ -17 | abs }}
{{ 4 | abs }}
```

`abs` will also work on a string that only contains a number.

{% raw %}
```
//input
{{ "-19.86" | abs }}
```
{% endraw %}

```
//output
{{ "-19.86" | abs }}
```
